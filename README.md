# kindle

まゆみのKindle出版プロジェクト。原稿・表紙・特典ページ等のマスターデータを管理します。

## 構造

```
kindle/
├── README.md                    - このファイル
├── status.md                    - プロジェクト進捗・UTAGE/メルマガ状況
├── kindle_cover_gem.md          - Kindle表紙制作メモ
└── <書籍タイトル>/              - 書籍ごとのマスターデータ
    ├── 01_本文/
    ├── 02_ペーパーバック/
    ├── 03_表紙/
    ├── 04_LINE特典ページ/
    ├── 05_配布特典/
    └── README.md
```

## 既刊

- **美容室に行くのに許可を取っていた私が、猫みたいに生きられるようになった話**
  - 2026年4月 出版
  - A5 / 80ページ / 約27,000字
  - ¥1,000（Amazon.co.jp）
